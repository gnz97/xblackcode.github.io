Aplikasi Simulasi Musik Saron

Aplikasi PEMILIHAN SEPEDA LISTRIK merupakan aplikasi sistem pendukung keputusan untuk menentukan sepeda listrik yang terbaik, berdasarkan kriteria-kriteria yang di tentukan, seperti Kecepatan, Harga, Jenis Daya dll, pada aplikasi sistem aplikasi tersebut menerapkan methode mora sebagai sitem pendukung keputusannya

Pada aplikasi tersebut terdapat bebrapa menu seperti :

  • Menu Home
  • Menu Panduan
  • Menu Profil
  • Menu Hasil Perhitungan
  • Menu Alternatif
  • Menu Kriteria
  • Menu Skala

Aplikasi Tersebut di buat menggunakan :

  • Template Admin LTE
  • Bootstrap
  • CSS/HTML
  • PHP
  • Codeigniter 3
  • Mysql
  • VSCode
